## 🚀💻 Projeto 
Esse é um projeto Web Responsivo de uma cápsola do tempo para exibir memórias em uma linha do tempo.

## 	🧑‍💻🚀 Tecnologias
Esses projeto foi desenvolvido durante o NLW da Rocketseat com as seguintes tecnologias: 

- HTML
- CSS
- GIT e Github
